# School of Discipleship

A modern Facilitator Management System built for church discipleship classes.

## Features

- Authentication
- Role-based Access
- Student Management
- Attendance
- Quiz Management
- Recitation
- Performance
- Physical Arrangement
- Group Chips
- Demerits
- Reports
- File Repository
- Audit Logs

## Tech Stack

Frontend
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui

Backend
- Node.js
- Express
- Prisma

Database
- PostgreSQL
- Supabase
