# User Manual — School of Discipleship

## Logging in
Go to the app URL → enter email/password → you land on the Dashboard.
Forgot password? Use the link on the login page.

## Navigation
The sidebar shows only what your role can access. Every list (Users, Students, Activities, PA, Reports) supports search, filters, and pagination.

## Common tasks
- **Take attendance**: Attendance → pick batch/class → New session → mark each student → Save draft → Submit & lock when done.
- **Grade an activity**: Activities → New activity → enter scores → Submit & lock.
- **Grade Physical Arrangement**: Physical Arrangement → New session → enter rubric scores per group (auto-computed total) → Submit & lock.
- **Award/deduct chips**: Physical Arrangement page → Group Chips panel.
- **Weekly checklist**: Weeks → pick batch → Add week → check off items → mark Completed (auto-locks).
- **Export a report**: Reports → pick report type + filters → CSV / Excel / PDF.

## Locking
Most records lock once submitted. Only a Super Admin can unlock a locked record, and must give a reason (recorded in the audit trail).
