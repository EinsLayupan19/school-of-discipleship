# Facilitator Guide (MDC / CC)

You only see data for classes you're assigned to facilitate — this is enforced by the system, not just hidden in the UI.

## Your workflow each Sunday
1. **Attendance** → New session (date must be a Sunday) → mark Present/Late/Absent per student → Submit & lock.
2. **Physical Arrangement** (if scheduled) → grade each group on the 5 rubric criteria.
3. **Weeks** → update the weekly checklist for your batch.
4. Award **chips** for good group performance if applicable.

## Watch for
- **Drop warnings**: a red badge appears once a student hits 3 absences. Check the notification bell (top right) for these and other reminders.
- **Locked records**: once you submit, you can't edit until a Super Admin unlocks it. Double-check before submitting.

## What you can't do
- Create/edit other users.
- See students, classes, or reports outside your assigned classes.
- Permanently delete a student (you can Archive instead — reversible).
- Unlock a locked record — request this from a Super Admin.
